### 🦿 Exoskeleton Digital Twin & Control System *(Bachelor Thesis — Cum Laude, 98%)*
Embedded control system for a bipedal exoskeleton in **C++ / RTOS**, with a real-time **Digital Twin in Unity (C#)** for medical and rehabilitation studies. Featured at the **Premio FRISA - UDEM 15th Edition** for best engineering thesis.

### 🤖 Digital Twin for Robotic Assembly Platform
Full pipeline from physical robot building to a **Unity-based Digital Twin** used to train and simulate robot behavior in an assembly line context.

Augmented Reality application for displaying and simulating historical mechanical objects. Built with **Unity + Blender**, in collaboration with an international team from Germany, Malaysia, Tunisia, and Oman.
